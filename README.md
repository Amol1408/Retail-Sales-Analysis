# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data to identify key trends, product performance, and revenue patterns using SQL and Power BI.

## Tools Used
- SQL
- Power BI
- Excel

## Key Metrics
- Total Revenue
- Sales by Category
- Sales by Region
- Monthly Sales Trend

## Dashboard Insights
- Electronics category generated the highest revenue.
- Sales increased during holiday seasons.
- Certain regions showed lower performance indicating improvement opportunities.
